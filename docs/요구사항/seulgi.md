# 슬기 요구사항 (프로젝트 리더)

## 꼭 필요한 기능

### 1. 가격 계산
- 입력 필드를 입고가, 출고가, 조합원가 3개로 분리해야 함 (현재는 입고가+조합원가 2개만 있음)
- 입고가: 연합에서 물건을 사오는 가격
- 출고가: 연합에서 단협에 넘기는 가격 (= 조합원가 × 0.74)
- 조합원가: 조합원에게 판매하는 최종 소비자가
- 세 가지를 모두 별도 입력할 수 있어야 하고, 출고가는 조합원가 입력 시 자동 계산(× 0.74)도 지원
- 슬라이더로 단협마진, 입고가 할인 조절
- 연합 마진 10% 이하 / 단협 마진 16% 이하 경고

### 2. 홍보 문구 (RCS 문자)
- 100자를 최대한 꽉 채워서 작성
- 이모지를 적극 활용 (🔥🎁✨💰⏰ 등)
- 줄 바꿈을 활용해서 가독성 좋게 구성
- 예시 형태:
  ```
  🔥[두레생협] 오늘만 특가!
  ✨ 데친문어(600g)
  💰 38,600원 → 32,800원 (15%↓)
  ⏰ 9시~19시, 한정 200개!
  ```
- 3가지 버전 자동 생성
- 복사 버튼 필수

### 3. 홍보 이미지
- 기존 timesale-card.html 스타일의 카드 이미지 자동 생성
- 상품 정보 입력하면 바로 미리보기
- 이미지 다운로드 기능

## 있으면 좋은 기능
- 이전 타임세일 기록 조회 및 재사용
- 쇼핑몰 팝업용, 앱 푸시용 문구도 함께 생성
- 여러 상품을 한 번에 등록하는 일괄 입력 기능
