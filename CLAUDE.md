# 타임세일 자동화 프로그램

## 팀원 (= 서비스 사용자)
- 지원: 프로젝트 리더 (디자인/개발)
- 소영: 팀원 (쇼핑몰 운영)
- 슬기: 팀원 (쇼핑몰 운영)

## 현재 단계: 니즈 파악 → 기획 → 설계 → 개발

## 프로젝트 목표
MD에게 타임세일 요청이 들어왔을 때, 가격 세팅부터 홍보 문구/이미지까지 자동으로 생성하는 도구를 만든다.

## 서비스 핵심 요구사항

### 1. 가격 자동 계산
- 입력: 상품코드, 상품명, 입고가, 출고가(조합원가), 한정수량
- 자동 계산: 연합 마진, 단협 마진, 할인율, 할인가
- 마진율 조절: 단협 마진 26% → 최소 16%까지 조정 가능
- 연합 마진: 10% 이하로 떨어지지 않도록 경고
- 입고가 할인: MD 협의에 따라 0~30% 조정

### 2. 가격 계산 공식
```
[정상가]
출고가 = 조합원가 × 0.74  (단협 마진 26%)
연합 마진 = 1 - (입고가 / 출고가)

[타임세일가]
할인 조합원가 = 조합원가 × (1 - 총할인율)
할인 출고가 = 할인 조합원가 × (1 - 단협마진)
  예: 단협 16% → × 0.84 / 단협 21% → × 0.79
할인 입고가 = 입고가 × (1 - 입고가할인율)
연합 마진 = 1 - (할인 입고가 / 할인 출고가)  ← 10% 이상 유지!
단협 마진 = 1 - (할인 출고가 / 할인 조합원가) ← 16% 이상 유지!
총할인율 = 1 - (할인 조합원가 / 조합원가)
```

### 3. 홍보 문구 자동 생성
- RCS 문자용: 100자 이내, 후킹 문구 3가지 버전
- 앱 푸시용: 50자 이내
- 쇼핑몰 팝업용: 간결한 문구
- 타임세일 기본 정보: 당일 오전 9시~저녁 7시

### 4. 홍보 이미지 자동 생성
- 기존 timesale-card.html 스타일의 카드 이미지 자동 생성
- 상품 정보 입력하면 바로 미리보기
- 이미지 다운로드 또는 URL 공유 기능

## 마진율 제약 조건 (절대 규칙)
- 연합 마진 ≥ 10% (위반 시 빨간 경고)
- 단협 마진 ≥ 16% (마지노선)
- 정상 단협 마진: 26%

# Claude 사용 가이드 (처음 쓰는 팀원을 위한 안내)

## 기본 사용법
1. 이 프로젝트 폴더에서 터미널을 열고 `claude` 입력
2. 말로 시키면 됩니다 (예: "가격 계산 기능 수정해줘", "홍보 문구 템플릿 추가해줘")
3. 작업이 끝나면 "커밋하고 푸시해줘" 라고 말하기
4. 작업 시작할 때 "pull 해줘" 라고 말하기

## 자주 쓰는 명령어 (Claude에게 이렇게 말하세요)
| 하고 싶은 것 | Claude에게 할 말 |
|---|---|
| 최신 코드 받기 | "pull 해줘" |
| 내 작업 저장 | "커밋하고 푸시해줘" |
| 팀원이 뭐 했는지 보기 | "최근 변경사항 보여줘" |
| 브랜치 만들기 | "feature/내이름-작업내용 브랜치 만들어줘" |
| 충돌 해결 | "충돌 해결해줘" |
| main에 합치기 | "main에 머지해줘" |

## 작업 순서 (매번 이렇게 하세요)
```
1. "pull 해줘"              ← 시작할 때 최신 내용 받기
2. 작업하기                  ← Claude에게 시키거나 직접 작업
3. "커밋하고 푸시해줘"        ← 끝나면 저장하고 올리기
```

## 주의사항
- 작업 시작 전에 반드시 pull 먼저!
- 같은 파일을 동시에 수정하면 충돌 날 수 있음 → 서로 작업 범위 공유하기
- .env 파일은 절대 커밋하지 않기 (비밀번호 등 포함)

# 프로젝트 규칙

## 문서 작성 규칙
- 문서는 한글로 작성
- 파일명은 영문 소문자, 단어 구분은 하이픈(-) 사용
- 날짜 포함 시: YYYY-MM-DD 형식 (예: 2026-03-20)

## 브랜치 규칙
- main: 완성된 코드만 (직접 수정 금지)
- feature/이름-작업내용: 각자 작업용 브랜치
  - 예: feature/jiwon-price-calc, feature/soyoung-promo-text, feature/seulgi-image

## 커밋 메시지 규칙
- 한글로 작성
- 예: "가격 계산기 UI 추가", "홍보 문구 생성 기능 구현"
